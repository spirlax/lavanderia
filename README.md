# Lavandería

Proyecto técnico de tesis para mejorar la gestión operativa de una pequeña lavandería.

## Estado

Fase 0: inicialización técnica y documentación preliminar. Todavía no existen conexiones a Supabase o Vercel, autenticación, migraciones ni módulos funcionales.

## Desarrollo local

```bash
pnpm install
pnpm dev
```

## Verificación

```bash
pnpm lint
pnpm typecheck
pnpm build
```

Los requisitos, decisiones preliminares y plan de fases están en `docs/`.
